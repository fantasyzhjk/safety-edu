# safety-edu

school 为学校 id

Usage:
  safety_edu [--hours=<t>] study <username> <password> [<school>]
  safety_edu [--score=<s>] exam <username> <password> [<school>]
  safety_edu info <username> <password> [<school>]
  safety_edu search <school>
  safety_edu (-h | --help)
  safety_edu (-v | --version)
